Patches

How to Patch via Battle.net

In most cases, it is best to use the auto-patch feature via the, in game, Battle.net connection. To do this, use the following steps.

  1. (Mac OS X users may skip this step.) Use the Extension Manager control panel to restart with the Mac OS Base extension set.

  2. Insert the Warcraft III: The Frozen Throne CD into the drive on your computer.

  3. Double-click on the Warcraft III application icon in the Warcraft III folder on your hard drive.

  4. Click on the Multiplayer option in the main menu screen for Warcraft III.

  5. Click on the Battle.net option in the Multiplayer menu screen.
If there is an update available, it will be downloaded and installed automatically, and the Warcraft III application will restart.




How to Patch Manually

In some cases, when problems occur with the Battle.net auto-patch feature, it may be necessary to manually patch the Warcraft III application. Use the following steps to manually patch Warcraft III.

  1. (Mac OS X users may skip this step.) Use the Extension Manager control panel to restart with the Mac OS Base extension set.

  2. Start a web browser application.

  3. Direct your web browser to either of the following sites. http://www.blizzard.com/patches
    http://ftp.blizzard.com/pub/war3/patches

  4. Download the most recent update for Warcraft III from either site.

  5. Decompress the update.

  6. Read the "How to Install" file included in the decompressed update folder.



How to Correct Patching Errors

To avoid patch problems, it is best to quit open applications other than Warcraft III before you start a Battle.net session. If you should run into an error with the automatic patch process, install the patch manually.

Patch errors occur when a patch file or a game file is corrupted. If you install a patch manually and you subsequently encounter an error when you attempt to start a Battle.net session, use the following checklist to troubleshoot the patch error.

  1. Delete the Battle.net Cache file. It is in the Warcraft III Files folder inside the Warcraft III folder on your hard drive.

  2. If you are using Mac OS 9, open the System folder on your hard drive. Open the Preferences folder. Then delete the Warcraft III Preferences file.

  3. If you are using Mac OS X, open your Home folder. Open the Library folder. Open the Preferences folder. Then delete the Warcraft III Preferences file.

  4. If the error persists, we suggest you back up any maps you have created and back up your character files. Then reinstall the game.
