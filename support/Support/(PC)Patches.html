Patches

How to Patch via Battle.net

In most cases, it is best to use the auto-patch feature built into the game via your Battle.net connection. To do this, use the following steps.

  1. Insert the Warcraft III: The Frozen Throne CD into the CD-ROM drive on your computer.

  2. Click on the "Start" button, select "Programs", select "Warcraft III", and select the "Warcraft III: The Frozen Throne" icon to launch the game.

  3. Click on the "Battle.net" option from the main menu screen.
If there is an update available, it will be downloaded and installed automatically. The Warcraft III: The Frozen Throne application will restart.



How to Patch Manually

In some cases, when problems occur with the Battle.net auto-patch feature, it may be necessary to manually patch the Warcraft III: The Frozen Throne application. Use the following steps to manually patch Warcraft III: The Frozen Throne:

  1. Start a web browser application.

  2. Direct your web browser to either of the following sites.
    http://www.blizzard.com/patches or http://ftp.blizzard.com/pub/war3x

  3. Download the most recent update for Warcraft III: The Frozen Throne from either site.

  4. Place the patch file into the Warcraft III game folder (where the game was installed on your hard drive) and double-click on the patch file to install it.



How to Correct Patching Errors

To avoid patch problems, please close any open applications other than Warcraft III: The Frozen Throne before you start a Battle.net session. If you run into an error with the automatic patch process, install the patch manually.

Patch errors occur when a patch file or a game file is corrupted. If you install a patch manually and you subsequently encounter an error when you attempt to start a Battle.net session, use the following checklist to troubleshoot the patch error.

  1. Delete Warcraft III: The Frozen Throne completely from the system and reinstall it. If you cannot load the patch automatically by connecting to Battle.net, download and install the patch manually.


  2. Make sure you have installed the latest version of DirectX as well as your video and sound card drivers. You should also install any critical updates Microsoft recommends on their update site. http://windowsupdate.microsoft.com


  3. Try running virus scanning software that is up to date to ensure you do not have a virus on the system. Check the Blizzard Entertainment Support site, under "Hacks, Warnings, and Security information" for information regarding virus software.
