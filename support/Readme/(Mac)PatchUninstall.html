Patching & Uninstalling

Patch Information

How to Patch via Battle.net:

In most cases, it is best to use the in-game, Battle.net auto-patch feature. To do this, use the following steps.

  • (Mac OS X users may skip this step.) Use the Extension Manager control panel to restart with the Mac OS Base extension set.

  • Insert the Warcraft III: The Frozen Throne CD into the drive on your computer.

  • Double-Click on the application icon for The Frozen Throne in the Warcraft III folder on your hard drive.

  • Click on the Battle.net option in the Main Menu screen.

If there is an update available, it will be downloaded and installed automatically, and The Frozen Throne will restart.


How to Patch Manually:

In some cases, when problems occur with the auto-patch feature, it may be necessary to manually apply a patch. Use the following steps to manually patch The Frozen Throne.

  • (Mac OS X users may skip this step.) Use the Extension Manager control panel to restart with the Mac OS Base extension set.

  • Start a web browser application.

  • Direct your web browser to either of the following sites.
    http://www.blizzard.com/patches
    http://ftp.blizzard.com/pub/war3x

  • Download the most recent update for The Frozen Throne from either site.

  • Decompress the update.

  • Read the "How to Install" file included in the decompressed update folder.



Uninstall Information

Warcraft III: The Frozen Throne is installed into a single folder on your hard drive, with the default name of "Warcraft III." When the game is first run, preferences files named "Warcraft III Preferences", "Battle.net Preferences", "com.blizzard.Warcraft III", and "com.blizzard.Warcraft III.plist" may be created on your hard drive. We do not install any Extensions or Control Panels. To uninstall the game, use the following steps.

For Mac OS X:

  • From the "Go" menu in the menu bar, choose "Home."

  • Open "Library" folder.

  • Open "Preferences" folder.

  • Throw away the "Warcraft III Preferences" file, the "Battle.net Preferences" file, the "com.blizzard.Warcraft III" file and the "com.blizzard.Warcraft III.plist" file.

  • Find the Warcraft III folder on your hard drive, and open it.

  • If you intend to reinstall, back up any custom maps you have created in the Maps folder.

  • If you intend to reinstall, back up any saved games you have created in the Save folder.

  • Close the Warcraft III folder.

  • Move the Warcraft III folder to the trash.

  • Empty the trash.

For Mac OS 9.2.2 and Earlier:

  • Open the System folder on your hard drive.

  • Open the Preferences folder.

  • Throw away the "Warcraft III Preferences" file, the "Battle.net Preferences" file, the "com.blizzard.Warcraft III" file and the "com.blizzard.Warcraft III.plist" file.

  • Close the Preferences folder.

  • Close the System folder.

  • Find the Warcraft III folder on your hard drive, and open it.

  • If you intend to reinstall, back up any custom maps you have created in the Maps folder.

  • If you intend to reinstall, back up any saved games you have created in the Save folder.

  • Close the Warcraft III folder.

  • Move the Warcraft III folder to the trash.

  • Empty the trash.
