Addendum

Manual Errata

Sound

  • Windows 2000 systems that use the EAX sound provider can experience extended pauses during play, and these pauses can cause sounds to loop. This problem will clear up after several seconds. You can avoid this issue by playing with the Miles Fast 3D or Dolby Surround sound providers.
Gameplay Changes

  • The F12 key now opens a message log history in single-player levels. This log tracks all your conversations with NPCs and any hints or quest updates you received.


  • Crates and gates can be destroyed by attack-clicking or right-clicking on them.
