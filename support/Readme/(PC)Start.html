Getting Started



Minimum System Requirements

Computer: A 400 MHz Pentium II or equivalent processor is required.

Software Requirement: Warcraft III: Reign of Chaos must be installed in order to install the Warcraft III: The Frozen Throne™ expansion set.

Operating System: Warcraft III is supported on the following operating systems: Windows 98, Windows Me, Windows 2000, and Windows XP.

Memory: Warcraft III requires 128 MB of RAM.

Controls: A keyboard and mouse are required. Joysticks, game pads, graphics tablets, and input devices other than a mouse and keyboard are not supported.

Drives: A hard drive with 700 MB of free space (in addition to your Warcraft III: Reign of Chaos installation - 1.4 GB total), and a 4x or faster CD-ROM drive are required for installation and play.

Video: A 3D accelerator video card with support for DirectX® 8.1 and at least 8 MB of video memory is required*.


*Please check http://www.blizzard.com/support/?id=mwr0555p to see if your video card is compatible.

Sound: A DirectX®-compatible 16-bit sound card is recommended.

Multiplayer Connectivity: Access to Blizzard Entertainment's online gaming service, Battle.net, requires a low-latency, active Internet connection, rated at 28.8 Kbps or faster. Multiplayer games played over a LAN require an active connection to a TCP/IP network.



Recommended System Requirements

Computer: A 600 MHz processor or better is recommended.

Memory: We recommend 256 MB of RAM.

Video: A 3D accelerator video card with support for DirectX® 8.1 and at least 32 MB of video memory is recommended*.

*Please check http://www.blizzard.com/support/video.html to see if your video card is compatible.

Sound: A 16-bit sound card that is compatible with DirectX® 8.1 is recommended.



Installation

To install Warcraft III: The Frozen Throne™, insert the Warcraft III: The Frozen Throne™ CD into your CD-ROM drive. If your computer is AutoPlay capable, a Warcraft III: The Frozen Throne™ splash screen will appear. Select "Install Warcraft III: The Frozen Throne™" from the list and follow the on-screen instructions.

If your computer is not AutoPlay capable, open the "My Computer" icon found on your desktop or Start button. Open the drive letter that corresponds to your CD-ROM drive; double-click the "Install" icon; and follow the on-screen instructions.

Note: If you have not installed Warcraft III: Reign of Chaos, you will be prompted to provide a valid CD Key for both Warcraft III: Reign of Chaos and Warcraft III: The Frozen Throne at install time. It is not possible to install Warcraft III: The Frozen Throne without also installing Warcraft III: Reign of Chaos.



Quick Start

Once Warcraft III: The Frozen Throne™ has been correctly installed, a program group will be created under the Start menu. Make sure the Warcraft III: The Frozen Throne™ CD is in the drive, and select the shortcut in the Warcraft III: The Frozen Throne™ program group to start the game. If your computer is AutoPlay capable, select "Play Warcraft III: The Frozen Throne™" on the splash screen that appears after the disc is inserted to start the game.
