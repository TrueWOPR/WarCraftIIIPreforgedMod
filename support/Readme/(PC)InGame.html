In-Game Controls

Basic In-Game User Interface

Alt-F4 or Alt-q - Exit Warcraft III.
Alt-o - Opens the Options Menu.
Alt-h - Opens the Help Menu.
Alt-s - Opens the Save Menu.
Alt-l - Opens the Load Menu.
F6 - Quick saves your game.
F10 - Opens/closes the Main Menu.
Ctrl-s - Toggles on/off Sound.
Ctrl-m - Toggles on/off Music.
Enter - Opens chat field to default setting.
Shift-Enter - Opens chat field to everyone in game.
Ctrl-Enter - Opens chat field to allies.
Print Screen - Takes a screenshot in .tga format and saves it to your Screenshots folder.
Esc - Takes you to a previous menu.
Insert - Rotates camera to the left.
Delete - Rotates camera to the right.
Mouse Wheel: Up - Zooms camera in.
Mouse Wheel: Down - Zooms camera out.
Mouse Wheel Button - Hold down and drag to adjust your view of the map.
Arrow Keys - Scrolls your view of the map.



Basic Game Commands

Mouse:

Left-click - Selects units, buildings, command buttons and points of action (where orders are to be carried out):

  • Selects units and buildings on screen.

  • Activates buttons from unit's command card.

  • Confirms placement of a command, building, ability, or item.

  • Uses an item from a Hero's inventory.

  • Trades or drops an item after it's picked up.

  • Changes game view to location on minimap.

  • Gives a subgroup focus when grouped with varying units.

Right-click - Performs "smart" actions on target units/buildings, ground, or minimap:

  • Ground - Moves unit to location, ignoring attacks.

  • Enemy Unit or Building - Attacks target.

  • Friendly Unit - Follows target.

  • Tree / Gold Mine - Moves unit to location (gathers resources if a worker unit).

  • Item on Ground - Hero moves to pick up item.

  • Item in Hero's Inventory - Picks up item.

  • When Targeting Cursor Is Active - Cancels command.
When Active Targeting Cursor Is Over the Minimap - Changes game view to location on the minimap.

Keyboard:

Esc - Cancels commands.
Tab - Switches between subgroups.
Alt (when held during a command) - Overrides formation movement during an action.
Alt-G - Minimap Signal. Click on the minimap to trigger a beacon for allied units.
Alt-T - Toggle minimap terrain.
Alt-A - Toggle ally colors.
Alt-F - Toggle formation movement. When this is on, melee units will stay in front of ranged units.
Ctrl (when held during a command) - Sends action to currently selected subgroup only.
Ctrl + # - Assigns selected units or buildings to a control group.
Shift + # - Adds selected units to active control group.
# - Selects assigned group. Two rapid presses centers view on the group.
F1-F3 - Selects corresponding Hero. Two rapid presses will center the view on a group.
F8 - Cycles through idle worker units. Each press will select and center on the next worker unit.
F9 - Opens the Quest Log screen. This allows you to view the status of each quest and optional quest.
F12 - Opens the Chat Log screen in multiplayer games, and opens the Message Log in single player games. The Chat Log is a record of messages sent to you by other players. The Message Log tracks all your conversations with NPCs and any hints or quest updates you received.
Backspace - Centers on your town center without selecting them.
Spacebar - Centers on the last 8 events that were shown on your minimap.
Alt-C or Ctrl-C - Centers on the currently selected unit.
Shift + Select Unit - Adds / Removes unit from current selection.
Ctrl + Select Unit or Double-click Unit - Selects all units of that type in the game view.
